# 💰 FinSphere AI

AI-powered personal finance management platform to track expenses, investments, goals, and financial insights in one place.

---

## 🚀 Features

- 📊 Dashboard with real-time financial insights
- 💸 Expense tracking with category analysis
- 📈 Investment portfolio management
- 🎯 Financial goals tracking
- 🤖 AI-powered insights
- 👤 Profile & settings management
- 🔐 Secure authentication (JWT)

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Lucide Icons

### Backend
- Node.js
- Express.js
- PostgreSQL

### Deployment
- Frontend: (Vercel / Render)
- Backend: Render
- Database: Render PostgreSQL

```bash

3. Setup Frontend
