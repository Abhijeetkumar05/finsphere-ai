# 💰 FinSphere AI

An AI-powered personal finance management system that helps users track expenses, investments, and financial health with smart insights.

---

## 🚀 Features

- 🔐 User Authentication (JWT based)
- 💸 Expense Tracking
- 📊 Dashboard with analytics
- 📈 Investment & Portfolio tracking
- 🎯 Goals Management
- 🧠 AI Financial Insights
- 📅 Smart filtering (date-wise)
- 🔒 Secure backend with PostgreSQL

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Chart.js / Recharts

### Backend
- Node.js
- Express.js
- PostgreSQL

### Authentication
- JWT (JSON Web Token)
